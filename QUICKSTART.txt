🔧 ЯКЩО ЩОСЬ НЕ ПРАЦЮЄ:

Проблема: Metro не запускається
  Рішення: npx expo start -c

Проблема: TypeScript помилки
  Рішення: npm install

Проблема: Reanimated не працює
  Рішення:
    rm -rf node_modules/.cache
    npx expo start -c

Проблема: Skia фон не рендериться
  Рішення: В Simulator → Hardware → Graphics Quality → High

Проблема: HealthKit не працює
  Примітка: HealthKit працює ЛИШЕ на реальному iPhone/iPad
            НЕ працює на симуляторі!
